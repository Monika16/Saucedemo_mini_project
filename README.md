# **Saucedemo**
This project demonstrates UI automation testing using:

- Python
- Pytest
- Playwright

The automation scenarios are implemented against the SauceDemo website:

https://www.saucedemo.com

# 🧪 Test Scenarios

## Login Tests
- Valid login
- Invalid login

## Product Tests
- Product visibility validation

## Cart Tests
- Add product to cart

## Advanced Scenarios
- JavaScript alerts
- iFrames
- Multiple browser tabs/windows
